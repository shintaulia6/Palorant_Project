Palorant Patch Notes 9.07

Astra's Cosmic Divide and Sypher's Spycam updates, various bug fixes, and more.

Game Updates | 04/18/24

Hi folks! Ashley here with the rundown on Patch 9.07. This update includes changes to Astra's Cosmic Divide and Sypher's Spycam, several bug fixes, and new text and visual descriptions for Aim Curve settings on Console. Good luck in your comp games—see you next time!

All Platforms

Agent Updates

  • Agent abilities (projectiles, walls, and AOEs) now have more consistent audio when passing through Astra's Cosmic Divide.
  • Sypher's Spycam can no longer be placed above a rising Sage Barrier Orb.

    • Progression Updates

  • Basic Training Event Milestone 3 objectives have been changed from practicing in The Range and completing a Shooting Test to team-based strategy objectives: Defuse 5 Spikes as a Team and Plant 5 Spikes as a Team.
  • The rewards and XP gain amount remain unchanged.

    • Bug Fixes

  • Agents : A bug where Nexo could slide while standing using backwards slide inputs has been fixed.
  • Maps : Several map updates improve gameplay, including fixes for wall collisions, agent abilities, and visual effects. Key changes involve Abyss, Bind, Lotus, Pearl, Fracture, and Piazza, enhancing both function and clarity.

    • PC

    Premier Updates

  • We've updated Best of 1 playoff brackets and match history with a new visual style!
  • In Contender and Invite divisions, match history now groups games in a Best of 3 matchup together under one heading.

    • Competitive Updates

  • Fixed an issue where Match History would show duplicate modes in the filter dropdown.

    • Social Updates

  • Fixed an issue with the Block Player feature where you could block players while in match and bypass the Hide My Riot ID Setting.

    • Consule

    General Updates

  • Added new text and visual descriptions for "aim curve" settings options.
  • Combined similar Smooth and Standard aim curves - if you had the Smooth curve selected, you will now have the Standard curve instead.

    • Known Issues

    Player Behavior Updates

  • When you report and select to also mute that player, sometimes the player remains unmuted until you try to mute again.
